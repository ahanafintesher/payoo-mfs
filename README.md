## ✨ Features

- 🔐 **Secure Login** — Mobile number + 4 digit PIN authentication
- 💰 **Available Balance** — Real-time balance display on dashboard
- ➕ **Add Money** — Add funds from bank account
- 💵 **Cashout** — Withdraw money with low charges
- 🔄 **Transfer Money** — Send money to other users
- 🎁 **Get Bonus** — Earn bonus rewards
- 🧾 **Pay Bill** — Pay utility bills easily
- 📊 **Transactions** — View transaction history
- 📱 **Responsive Design** — Works on all screen sizes

---

## 📦 Dependencies

- [Tailwind CSS](https://tailwindcss.com/) — Utility-first CSS framework
- Vanilla JavaScript — No framework needed
